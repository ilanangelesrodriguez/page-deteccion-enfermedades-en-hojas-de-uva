Descripción del Proyecto

Este es un proyecto de aplicación web utilizando Flask para cargar imágenes y mostrar resultados de un modelo de clasificación de imágenes. Las imágenes cargadas se almacenan en una base de datos PostgreSQL y el modelo de clasificación de imágenes se carga desde el archivo HDF5 (mi_modelo.h5).

El modelo fue entrenado utilizando el siguiente Dataset de Kaggle. Para el entrenamiento, se utilizó una red neuronal convolucional (CNN) debido a su eficacia en tareas de clasificación de imágenes.

Desarrollado por Ángeles Rodríguez Ilan, Loma Aguirre Mariesther
